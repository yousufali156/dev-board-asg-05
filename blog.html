Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM using JavaScript: Like this By Id, getElementById, By Class Name, By Tag Name, By Name Attribute, By Query Selector etc being the most versatile method.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML renders HTML content, innerText shows only visible text, and textContent retrieves all text, including hidden content, without rendering HTML

Question-3: What is event delegation in the DOM?

Event delegation allows a parent element to handle events for its child elements using event bubbling, improving performance and simplifying code, especially for dynamic content.

Question-4: What is event bubbling in the DOM?

Event bubbling starts from the target element and moves up through parent elements, enabling efficient event handling from the parent element.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, use document.createElement(). Add it with appendChild() or append(), and remove it with removeChild() or remove().
